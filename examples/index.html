Earlier 3D human body models mainly focused on representing different body shapes but had difficulty handling realistic pose-related deformations. Models such as SCAPE introduced pose-dependent shape changes but were computationally expensive and incompatible with standard graphics pipelines.

The SMPL (Skinned Multi-Person Linear) model addressed these limitations by combining pose-dependent deformations with linear blend skinning, resulting in a model that is efficient, realistic, and compatible with existing rendering engines.

Introduction

Creating realistic animated human body models has been a long-standing challenge in computer graphics. Linear Blend Skinning (LBS) is widely used but produces artifacts such as the candy-wrapper effect.

Model Overview

SMPL starts from a template mesh with N = 6890 vertices and K = 23 joints.

  • Single neutral template mesh
  • Fixed topology across identities
  • Compatible with standard animation pipelines

Mathematical Formulation

The mean template shape is \mathbf{T} \in \mathbb{R}^{3N}. Shape blend shapes are defined as:

B_s(\boldsymbol{\beta}) : \mathbb{R}^{|\boldsymbol{\beta}|} \rightarrow \mathbb{R}^{3N}

Pose-dependent corrections are captured by:

B_p(\boldsymbol{\theta}) : \mathbb{R}^{|\boldsymbol{\theta}|} \rightarrow \mathbb{R}^{3N}

Linear Blend Skinning

\mathbf{t}_i' = \sum_{k=1}^{K} w_{k,i} \mathbf{G}_k(\boldsymbol{\theta}, \mathbf{J}) \mathbf{t}_i

SMPL preserves standard LBS and adds corrective deformations in the rest pose, maintaining efficiency and compatibility.

Conclusion

SMPL achieves a balance between realism, efficiency, and usability by learning pose-dependent deformations while retaining standard skinning methods.
